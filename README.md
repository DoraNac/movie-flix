🎬 Meowflix

A simple Netflix-style movie application built with React and Vite. Users can search for movies and view information about their favorite movies.

Features

    🔍 Search Movies
    📚 Browse Categories
    📜 Movie Details

Tech Stack

    Frontend: React, Vite
    API: The Movie Database (TMDb)
